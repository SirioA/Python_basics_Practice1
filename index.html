Exercise 1: Variables and Basic Operations

Create variables for your name and age, then print a message introducing yourself.

Hint: Use quotes for strings and no quotes for numbers. Try using f-strings for formatting!
Sample Solution:
name = "Alice"
age = 20
print(f"Hello, my name is {name} and I am {age} years old.")

Exercise 2: Lists and Loops

Create a list of your favorite colors and print each color with its position in the list.

Hint: Use enumerate() to get both index and value, or use range(len(list))
Sample Solution:
colors = ["red", "blue", "green", "purple"]
for i, color in enumerate(colors, 1):
    print(f"{i}. {color}")

Exercise 3: Functions

Write a function that takes a list of numbers and returns the sum of all even numbers in the list.

Hint: Use the modulo operator (%) to check if a number is even. number % 2 == 0 means it's even!
Sample Solution:
def sum_even_numbers(numbers):
    return sum(num for num in numbers if num % 2 == 0)

# Or the longer version:
def sum_even_numbers(numbers):
    total = 0
    for num in numbers:
        if num % 2 == 0:
            total += num
    return total

Exercise 4: Challenge - Word Counter

Write a program that counts how many times each word appears in a sentence.

Hint: Use split() to break the sentence into words, and a dictionary to count occurrences.
Sample Solution:
sentence = "the quick brown fox jumps over the lazy dog the fox is quick"
words = sentence.split()
word_count = {}
for word in words:
    word_count[word] = word_count.get(word, 0) + 1

for word, count in word_count.items():
    print(f"'{word}': {count}")
